# Commands
<h3>ins</h3>
<i>Struct</i>
```
ins Text
```

<i>Description</i>
Output Text

<h3>set</h3>
<i>Struct</i>
```
set name = value
```

<i>Description</i>
Set Variable by name and value

<h3>varcheck</h3>
<i>Struct</i>
```
varcheck name
```

<i>Description</i>
Check variable information
